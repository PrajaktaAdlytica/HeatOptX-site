# HeatOptx Website

Premium SaaS website prototype for HeatOptx, an Industrial Thermal AI platform
that maps hidden heat loss, detects thermal anomalies, and ranks fixes by ROI.

## Included

- Homepage with product-led scroll storytelling
- HeatOptx Map, Detect, and ROI product sections
- Industrial problem framing and solution cards
- Request ROI assessment form
- Polished sign-in page
- HeatOptx logo, favicon, icon set, and illustration assets

## Local Development

```bash
npm install
npm run dev
```

## Validation

```bash
npm run lint
npm test
```
